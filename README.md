# This is a sample Cloudlfare website

## Install
1. install nodejs
2. install npm
3. install wrangler

## Usage
1. clone this repo
2. cd into the repo
3. run `npm install`
4. run `wrangler dev` to start the local server

## Configuring
1. change the `account_id` in `wrangler.toml` to your own
for example: 
route = "https://whatiswrongwithmycode.com/*"
